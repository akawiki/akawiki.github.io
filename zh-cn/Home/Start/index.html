电脑配置要求

游戏运行需要安装Unity Web Player，另外由于游戏本身提供的32位微端极易崩溃，强烈建议使用64位模式运行游戏。

IE用户开启64位模式

  • 下载开启64位工具
  • 按照压缩包说明操作（安装64位Unity Web Player以及设置IE）
  • 打开游戏登陆页面登陆游戏

非IE用户开启64位模式

  • 下载官方32位微端并安装
  • 打开游戏登陆页面登陆游戏并成功进入游戏一次（到晓轨的蓝天白云欢迎画面即可）之后关掉游戏
  • 下载开启64位工具
  • 按照压缩包说明操作（安装64位Unity Web Player以及将官方32位微端替换为64位）
  • 打开游戏登陆页面登陆游戏

若成功开启64位模式，整个游戏画面的右下角会出现如下标记：

游戏主页及登录页面需要翻墙，请自行准备翻墙工具（例如岛风GO）。

游戏本体固定直连服务器（即使使用chrome等浏览器启动微端）（另外直连游戏服务器并不需要翻墙），如果想要游戏获得VPN提速效果请使用Proxifier等软件。

游戏缓存

游戏缓存默认保存在以下位置，也可直接通过此快捷方式打开到缓存文件夹或下载保存使用。注意此文件夹需要进入过游戏才会出现。

系统盘:\Users\<用户名>\AppData\LocalLow\Unity\WebPlayer\Cache

此缓存文件夹为所有Unity Web Player游戏的文件夹，其中The Legend of Heroes - Akatsuki no Kiseki文件夹即为晓轨的缓存文件夹。

官方通常会在每周三的 中国时间(UTF+8) 10:00~14:00、日本时间(UTF+9) 11:00~15:00 （具体时间变动请参考官方主页）维护更新游戏服务器，期间不能进入游戏

维护后更新过的游戏数据会重新下载到缓存文件夹，也就是说老的数据会一直留在缓存文件夹中，导致此文件夹越来越大，所以建议每周三更新清一次缓存，如果网络实在很慢不愿意每次都重新下载没更新的缓存就不需要清空。

另外如果出现无法进入游戏或出现各种错误，可以尝试清除缓存文件夹，以及浏览器缓存（重要）

咪西（みっしぃ）

经验猫

金猫银猫铜猫，通称经验猫，用于在训练中提升人物等级

绿猫彩猫

用于在觉醒中提升人物星级

红猫蓝猫

用于在训练中提升人物战技等级

黑猫

用于在究极觉醒中提升人物等级上限

训练

提升人物等级的主要方式

消耗经验猫和米拉

虽然也可以吃有等级的其他人物，但是极度不推荐

战斗中击杀敌人虽然也可以获得一定的经验，但是效率太低也不推荐

训练的时候每使用一名同名角色能获得一次等同于红猫的随机战技等级提升效果

在训练中每使用一只红猫能随机提升一次技能等级，但是仅限于已经解放了的技能

像下图所示的人物由于只有2星，并未习得樱花斩与土龙烈波，此时若使用红猫进行训练只能在居合斩与S技之间随机提升一项

蓝猫的作用比较特殊，在人物达到6星时用于解放Command技，即指挥技

指挥技不同于其他技能在达到相应的星级后会自动解放。而是必须在达到6星后使用蓝猫习得

当人物达到6星且指挥技未习得时，使用蓝猫，100%会习得1级指挥技

其他情况使用蓝猫效果相当于成功率只有30%的红猫，成功时效果等同红猫；失败时扣除消耗的蓝猫与相应的米拉，然后不产生任何变化鲤鱼王使用了水溅跃！然而什么都没有发生

战技等级为同名人物共通，即如果你喂满一人，卡池抽出一个同名新人，战技等级也是满的

觉醒/究极觉醒

当一个人物达到30级满级时，就可以进行觉醒（升星）

1星30级觉醒需要1个1星狗粮（等级不限）

2星30级觉醒需要2个2星狗粮

…

5星30级觉醒需要5个5星狗粮

越高的星级觉醒成功率越低

但是只要加入一只彩猫或者一只本体就能让成功率保持在100%

但是极度不推荐使用本体做狗粮升星，下面的究极觉醒会说明

当一个人物达到6星30级时就达到星级上限了，此时需要进行的就是究极觉醒（左上觉醒右边，别问我为啥要提醒）

究极觉醒需要人物达到6星30级，通过吃黑猫或者6星本体提升2级等级上限。在提升上限后仍需要在训练中喂经验猫提升等级，当再次满级时才可进行下一次究极觉醒

一个人物可进行5次究极觉醒，即最高可达到40级

在使用同名人物进行觉醒/究极觉醒时，每个同名人物还会附带一次红猫的随机技能提升效果

由于黑猫较难入手，抽到的4,5星同名人物请务必只用在究极觉醒中

衣装觉醒

衣装觉醒属于特殊的觉醒，只有在究极觉醒中才能获得特装效果。虽然战技等级是人物共通的，但是特殊服装人物中的部分战技会发生改变，在切换服装后需要对改变的战技重新喂红猫以提升技能等级。

另外衣装觉醒无论特殊服装人物是作为本体还是狗粮，在究极觉醒后都能进行换装。可是只有本体的熟练度能保留，而熟练度并不共通，这方面需要留意。

装备/熟练度

装备阶级从高到底依次为 橙 紫 蓝 绿 白


后缀的优劣依次为

武器 咒＞祝＞良＞无后缀＞锻

防具 咒＞祝＞改＞无后缀＞锻


其中咒无法炼化进阶

祝消耗的材料为普通的1.5倍，但是炼化后一定保有祝后缀

锻作为强化狗粮有1.5倍的经验加成

强化到+5的狗粮也有1.5倍的经验加成，可与锻叠加


每个装备可强化至+5，+5后可进行炼化进阶

炼化之后颜色上升为高一阶，强化进度变为+0，且装备类型与后缀会发生随机变化，物理与魔法类别并不会发生改变


例如 +5的蓝色STR拳套良 在炼化后可能会变成 +0 的紫色 STR 刀剑锻


装备还有装备次数的限制，除了白色装备以及专属武器可无限制装备/卸下，其他当装备次数归零后可消费250BC重新装备

拥有「先制攻击」「奇袭攻击」效果的核心回路装备在后备人员身上也会起效

这5个都完成后建议把3个 遠征準備室 开出来 下面简单介绍下用途

キューブ保存室

用于将4色属性石转化成BC，无氪玩家主要的BC来源

会計室/会計室EX

作用就是，生产米拉。游戏对米拉的消耗非常巨大，很多时候你最缺的就是米拉

EX相比会計室功能更强，但是要做的事情基本差不多，因为米拉的高需求最好是同时运作。

而EX独有的功能是营业，即将少量的米拉转化为IBC证券，然后再通过IBC一部株式投資 售卖为米拉。

セピス生産室

晶片生产室，上位晶片（时空幻）的主要来源之一。新人只要记住，3小时做时片不要停

資材加工室

升级飞船需要的 一般板金 只能从这里制造，同时属性石除了刷图出以外也可以在这里消耗晶片与初级强化石制造

遠征準備室

在探索地的最后一张图达成S后即可以去那里进行远征获取少量资源

每张图都会标注有推荐战力，建议远征队伍的战力达到推荐战力的1.5倍，否则很可能远征不到有用的素材。具体地图能远征到什么东西请点击探索地选择右侧的按钮进行查看

远征室需要把15%的掉率堆满

第一远征室 需要战力评价数值

第二远征室 需要STR数值

第三远征室 需要ATS数值

战斗篇

战斗模式与队伍构成

晓之轨迹的战斗系统与轨迹系列完全一致，只在小部分有些出入。与零之轨迹最为接近。这里仅介绍页游版特有的要素。

界面基本与零轨一致，将观察与待机放到了右下角，交代是用来替换后备人物上场。右下手动的圈可在 手动-半自动-自动 3种模式间切换。

手动即为全程玩家控制

半自动为人物只会进行最基础的普通攻击

自动为人物会根据AI自行选择普通攻击或者释放战技与魔法

※半自动与自动都不会 撤退，使用道具，人物倒下时替换后备

一个队伍最多可容纳8人，战斗中最多4人在前场，另外4人作为后备待机

晓之轨迹的人物战技与单机不同之处在于存在 指挥技（Command） 与 被动技（Passive）

指挥技 必须人物处在前场时才会发动，并且当人物处于战斗不能状态中也会同时失效

被动技 只要人物在上述8人编队中就会一直发动效果


由于人物培养的难度问题，后备人员很少会换到前场进行战斗，此时后备人员最重要的作用就是提供覆盖全队增益效果的被动技（Passive）

以礼物·诺艾儿举例来说，她的被动3级满级效果是 对己方全体有效 DEX+40、STR+20%、受到伤害时AT+15/CP+5。也就是只要她在队伍中，上场的人物会极高的提升命中（40点的DEX），提升STR（20%），不论是被普通攻击，战技，还是魔法打中（必须造成伤害且大于0）AT会提前15并额外获得5点CP。15点的AT提前在人物有着高SPD的情况下约等于直接多动一次，而本身受伤害会获得15点CP，在这个被动的加持上能获得20点，很多单体技能（比如丽霞的雷暴符）就可以直接使用。

不过我们平时更多的关注点还是+ATS%，+STR%，+伤害%的被动。

游戏内计算公式

这里就要先介绍下晓之轨迹内部的一些公式算法了，首先是没什么卵用的 战力计算公式：

战力评价= [(MaxHp * 4) + (Str * 10) + (Ats * 10) + (Def * 5) + (Adf * 5)] * Spd)/ 200

可以看出最大HP和SPD影响的比重是最大的，血厚的人战力数值会特别高

然后是对战中的 伤害计算公式：

最终伤害=（基本伤害+随机伤害）* 被动伤害加成 * 暴击1.5倍加成 * 魔法有效率

基本伤害=[STR或ATS*(属性有效率/100)*(威力/100)*5-DEF或ADF*2.5)]/2

随机伤害=基本伤害/15

※魔法有效率 仅在 使用导力魔法（Arts）时参与计算。可以用战斗中右下轮盘的观察进行查看，若显示???需要用侦查魔法或者战斗探测器（解析胶囊）进行解析

※基本伤害中的STR或者ATS只有一项起效，具体根据人物属于物理还是魔法决定，而DEF或ADF根据起效的是物理还是魔法 对应单独起效

※属性有效率参考下图，克制关系为150，被克50，无直接对应关系的都为100

※威力可以在工具箱的人物战技内查询，普通攻击威力为100，CP200时释放S技在最终伤害时有1.5倍加成

捎带一提的DELAY计算公式：

DELAY=硬直时间 * 100/SPD

※硬直时间同样可在工具箱内查看

最后是 战斗中即时的STR/ATS 数值计算公式：

在这里借用C语言中的ceil函数。ceil(x)返回的是大于x的最小整数。如：

ceil(100.1) = 101

ceil(100.9) = 101

ceil(-2.5) = -2

实际战斗中的STR/ATS=ceil（人物裸值*被动%）+ceil(人物裸值*魔法BUFF%)+ceil(人物裸值*Command%)+ceil(人物裸值*公会提升%)+核心回路+武器防具（绝版防具还有+20）+熟练度

被动有上限50%，魔法BUFF有上限50%（就是顶头上有回合数限制的）

实际情况就是被动堆到50%就行了再多没用，但是command和公会可以无限堆（虽然command你也只能上4个，公会最多就10%）

※人物裸值为不含熟练度、回路、武器装备、各种BUFF的情况。可以点击能力检索查看，但是能力检索是包含了默认1级熟练度的情况，需要自行减掉。

由于被动伤害加成不存在上限，我们可以明白堆被动的基本原则是在堆满50%的STR/ATS被动的基础上，尽可能的堆 伤害增益类被动


以玲举例，玲的被动拥有提升自身30%ATS的能力

那么就需要以下人物来提供被动以达到最高输出了

瓦吉，己方全体20%的ATS提升与玲加一起刚好达到50%

缇欧，己方全体20%的伤害提升

公主，己方全体20%的魔法伤害提升

露西奥拉，己方全体25%的导力魔法伤害提升

小丑，己方全体20%的伤害提升

在队伍中加入以上5名角色后，玲在战斗中就拥有了50%的ATS提升，使用导力魔法时拥有85%的伤害提升。而且还有2个空位可以配置增加全体SPD的雪拉，增加暴击率的缇妲。

相对的物理队也是这样配置队伍，另外还需要兼顾命中，诺艾尔和魔弓作为后备是较好的选择。


※特例

由于码农的消极怠工，文本表述的不严谨等，导致部分技能在实战中出现与通常情况效果不相符的地方（或许以后会修正）。

又或是一些官方并没有解释，但在测试中得出结论的地方，在此列出：


①公主的被动技能只对导力魔法（Arts）有效，普通攻击与战技无效。这点与 冲突。

②暴击率存在上限，推测在70%~80%之间。

③每个人能从全体AGL被动加成获得的值最大为30点（公会，魔法BUFF，自身提升不包含在30点内）

例，40级的怪盗AGL裸值为38，队伍中存在瓦吉、杜巴莉、凯文时本应获得55点AGL加成，实际上怪盗的AGL只能到68

④伤害提升类被动不存在上限（比如魔法队在前文的举例中再带上一个露西奥拉好友能达到110%的导力魔法伤害加成）

⑤杜巴莉的豪炎剑、豪冰剑、豪雷剑与其他物理人物不同，同时享受导力魔法的魔法有效率加成（折损）。所以逗巴莉可以说是土系人物的天敌

其它

34级师徒小号速成

虽然不是写给新人看的，但是难点的攻略方式是共通的。

首先新人要学会在地图中撤退（不是战斗中那个撤退），前期因为刷图的经验非常多导致AP溢出十分严重。撤退能让你在不增加分队经验的前提下获得之前刷的所有东西。当然如果你是条AP都用不完的咸鱼，就跳过上面那段话吧……


好了开始正题，这个一日速成也不是从创建号开始算起的，需要一些准备工作，比如先签到7天拿到罗爷艾丽。然后我们就可以开始了。什么？你问为什么要刷小号，那当然是为了刷任意卷了，一个34级小号能产出20张任意卷。不过讲真这效率真的很低，花的力气太大，不如氪一单，不出货也有30多张任意了。

队伍构成由罗爷 艾丽 序章训练送的雪拉 第一个10连给的4星组成（2250BC打主线自然就攒够了）。拿到的经验猫有多少吃多少，等级平均分配即可，熟练度强化优先给罗爷，至少都点到需要消耗铜或亚麻。结晶孔在保证每个人装备6个1级回路的前提下尽可能全开，优先开艾丽的，普通难度的序章训练全程自动都能过，可能会死人，不过这并不重要。

主要战术为罗爷顶前面卖血放S技，艾丽用属性弱点魔法攻击，另外2个人加血。还有墙角卡尸体的战术后文再解释。

由于前期AP溢出十分严重，但是速成并不需要花时间把多余的AP转化为素材，可是中期又会因为AP不够清不完训练。所以前期的AP优先清训练，然后走主线，尽量控制AP的溢出。

第一个难关，一章-普通-30。由于队伍很弱无法有效输出，而BOSS一个S技就秒人。这就需要一些战术了，罗爷一个人顶上去吸收攻击，回复术奶自己，情愿站着不动待机也不要去攻击敌方送CP，等CP100了抢HP回复奖励或者暴击输出。因为克鲁兹是弱幻的，艾丽装备EP1和精神1就能出单体的混沌烙印，雪拉全程加血。此时可能没有第一个10连，或者第一个十连抽出的是罗爷艾丽雪拉，并不影响通关，本体可以留着后面升5星当彩猫。

另外如果在前面 序章 训练卡壳的玩家也可以用类似的打法磨过去。

之后继续推图，记得加自己大号好友或者抱个大腿。序章剧情池不抽。


一章-困难-8。普通难度下随便打打就过了，困难难度下那空牙稳定秒杀艾丽，所以站位调整至如下：

因为狼不会放S，所以罗爷尽量用转圈嘲讽控制，其他人用水系魔法输出，如果罗爷能封技住是最好的结果，优先以保住罗爷为主。其他人最好都能配出复活魔法，商店有3级回路直接买。本攻略对BC的消耗只有第一个10连。

恐怖的番茄，速度快，各种连动。这个时候就要借助卡尸体大法了。

由于大番茄占地3X3而且只能近战，所以罗爷如果在上图位置死亡就会成为一道无法移动的路障，躲在内部的人员就可以慢慢磨死大番茄，只是要注意别让番茄吃了50HP之类的。

之后一路推图到利贝尔王都，打完一章主线后就可以抽剧情池了，如果运气好是可以抽干剧情池获得5星克鲁兹的，这样打一章-困难-11就更方便了。没有的话也不要紧，抽出来的狗粮是够做2个5星的，推荐把雪拉和罗爷升到5星，前者的SPD被动能让你全队行动更快，后者还是用来磨训练关卡的。攻略途中还可以刷3把限时绿猫本或经验猫本，出现时间请自行查询wiki。当做出2个五星甚至3个五星后攻略就变得简单起来了。

或许是最后一个难关，二章-困难-1、2，开场敌方先制行动释放魔法。我方4角站位即可，如果不行还可以事先在配置队伍时把狗粮放前面，战斗不能后换主力上场。

如果顺利通过了前面这些训练关卡，那么后面直到34级都没什么难度了。想要一天刷到34级对AP的分配十分严格，笔者也是尝试了3次才做到18小时升到34级。就结果来说效率十分低，当成挑战自我姑且还可以一试。


攻略就到此为止了，人物强度可以参考其他大佬的评测及节奏榜，笔者不做任何评论（撕逼）
